Inline Math Support using MathJax

  • Put your math expression within $\$...\$$
  • eg $\$$x_1 + 2$\$$ will be render as $x_1 + 2$

Implementation.

  • I used marked to convert markdown code to html code.
  • I used MathJax to render math expression in LaTex style.
  • I used JsPDF to compile rendered html to pdf file.
